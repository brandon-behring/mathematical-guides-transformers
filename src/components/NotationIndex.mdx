**Column-vector switches.** Rows of $\mathbf X\in\R^{n\times d}$ are the default. The following scopes temporarily use columns so left-acting matrices, Jacobians, or outer products have their conventional orientation.

| Scope | Objects treated as columns | Reason |
|---|---|---|
| Softmax Jacobian | $\mathbf z,\mathbf p$ | $\mathbf J$ acts by left multiplication and $\mathbf p\mathbf p^\top$ is an outer product |
| LayerNorm/RMSNorm Jacobians | $\mathbf x,\hat{\mathbf x},\boldsymbol\gamma,\mathbf 1$ | normalization acts on token rows, but Jacobians act on columns and use outer products |
| Positional-rotation derivations | RoPE queries/keys and 2D rotation pairs | rotations act as $\mathbf R\mathbf x$; transpose returns to row form |
| Recurrent Networks | inputs, hidden states, and gate states | time Jacobians act by left multiplication |
| Linear Recurrences and SSMs | state $\statevec$ (with $\inputmat$ a column and $\outputmat$ a row) | standard control-theory orientation |
| Linear-attention derivation | $\phi(\mathbf k_j),\mathbf v_j$ | $\phi(\mathbf k_j)\mathbf v_j^\top$ is a $D\times d_v$ outer product |
| MoE gradient calculus | gradient vectors | $\nabla_{\mathbf W_r}\mathcal L=\mathbf x^\top(\nabla_{\mathbf s}\mathcal L)^\top$ |
| Selective State Spaces | state/write/read vectors; feature/value vectors in the linear-attention subsection | left-acting recurrence and matrix-state outer products |
| Modern Recurrent & Hybrid | query, key, value, and recurrent-state vectors | outer-product state updates |
