# The Attention Mechanism

**Where we are.** Attention is the transformer's one genuinely new operation: each token aggregates information from every other token, selectively. We build it from one nonlinearity (softmax), one similarity (the dot product), and one piece of normalization (the $1/\sqrt{d_k}$ scaling), and prove the property that makes each true.

## Notation

Vector conventions, the standing dimensions ($n$, $d$, $d_k$), $\defeq$, the inner product, and the softmax are all fixed in the notation chapter; we invoke them by name. This chapter introduces the attention operator $\operatorname{Attn}(\mathbf Q,\mathbf K,\mathbf V)$, the causal mask $\mathbf M$, and the per-head value dimension $d_v$.

## Softmax, recalled

Softmax turns a vector of scores into a probability distribution: larger scores get exponentially more weight, and the weights sum to one. Its definition and its two working properties are fixed once in the notation chapter — the definition (<XRef id="def-softmax" />), translation invariance (<XRef id="prop-softmax-translation-inv" />), and the Jacobian (<XRef id="prop-softmax-jacobian" />). Everything this chapter proves rides on the first two: the definition makes each attention weight row a probability vector, and translation invariance is what makes the numerically stable max-subtracted implementation exact.

## Scaled dot-product attention

Structurally, attention is **Nadaraya–Watson kernel regression with a learned kernel**: each output is a kernel-weighted average of the value vectors, where the weight value $j$ receives is proportional to $\exp(\inner{\mathbf q_i}{\mathbf k_j}/\sqrt{d_k})$. The correspondence is exact — *kernel weights* ↦ *a softmax row*, *kernel-regression estimate* ↦ *the attention output* — and it breaks in exactly one place: the kernel is **learned**, so it need not be symmetric or positive-definite, and the classical kernel-smoothing guarantees do not transfer. We make the construction precise.

<Theorem kind="definition" name="Scaled dot-product attention" id="def-tf4-attention">
Given queries $\mathbf Q\in\R^{n\times d_k}$, keys $\mathbf K\in\R^{n\times d_k}$, and values $\mathbf V\in\R^{n\times d_v}$,
$$
\operatorname{Attn}(\mathbf Q,\mathbf K,\mathbf V) \defeq \softmax\!\Big(\tfrac{\mathbf Q\mathbf K^\top}{\sqrt{d_k}}\Big)\,\mathbf V,
$$
with softmax applied row-wise to the $n\times n$ logit matrix.
</Theorem>

Because each softmax row is non-negative and sums to one (<XRef id="def-softmax" />), the weight matrix is row-stochastic, so output row $i$ is a **convex combination** of the value rows (<XRef id="prop-row-stochastic-convex" />) — the kernel-regression reading made literal.

For language modeling, attention must additionally respect time: position $i$ may consult only positions $j \le i$. This is imposed not by changing the operator but by adding a fixed mask to the logits before the softmax.

<Theorem kind="definition" name="Causal (masked) attention" id="def-tf4-causal-mask">
The **causal mask** $\mathbf M \in \R^{n\times n}$ has entries
$$
M_{ij} \defeq \begin{cases} 0 & j \le i,\\ -\infty & j > i,\end{cases}
$$
and **causal attention** is attention on the masked logits:
$$
\operatorname{Attn}^{\text{causal}}(\mathbf Q,\mathbf K,\mathbf V) \defeq \softmax\!\Big(\tfrac{\mathbf Q\mathbf K^\top}{\sqrt{d_k}} + \mathbf M\Big)\,\mathbf V.
$$
A $-\infty$ logit exponentiates to $0$, so each softmax row remains a probability distribution, now supported on $\{j : j \le i\}$: row $i$ attends only to positions up to $i$.
</Theorem>

## Why the $1/\sqrt{d_k}$ scaling

Each logit $\inner{\mathbf q}{\mathbf k}$ sums $d_k$ independent product terms, so its spread grows with $d_k$ and pushes softmax toward saturation. The scaling is chosen to *exactly* cancel that growth.

<MarginNote variant="tip" label="Self-explanation">
Before reading: $\inner{\mathbf q}{\mathbf k}=\sum_m q_m k_m$ adds $d_k$ independent zero-mean unit-variance terms. What should its variance scale like in $d_k$?
</MarginNote>

<Theorem kind="theorem" name="Variance preservation" id="thm-variance-preservation">
Let the $2d_k$ scalar components $q_1,\dots,q_{d_k},k_1,\dots,k_{d_k}$ be **mutually independent** with $\E[q_m]=\E[k_m]=0$ and $\Var(q_m)=\Var(k_m)=1$. Then $\E[\inner{\mathbf q}{\mathbf k}]=0$, $\Var(\inner{\mathbf q}{\mathbf k})=d_k$, and $\Var\!\big(\inner{\mathbf q}{\mathbf k}/\sqrt{d_k}\big)=1$.
</Theorem>

<Theorem kind="proof">
Three steps, each naming the rule it uses.

**Mean.** By linearity and independence,
$$
\E\Big[\textstyle\sum_{m} q_m k_m\Big] = \sum_m \E[q_m]\,\E[k_m] = 0
\qquad\text{(linearity; } q_m\perp k_m\text{)}.
$$

**Variance.** Since the mean is $0$, $\Var(\inner{\mathbf q}{\mathbf k}) = \E[(\sum_m q_m k_m)^2]$. For $m\neq n$ mutual independence factorizes the cross term into $\E[q_m]\E[k_m]\E[q_n]\E[k_n]=0$ — this is where independence *across* indices is used, not just $q_m \perp k_m$ — leaving
$$
\sum_m \E[q_m^2]\,\E[k_m^2] = \sum_{m=1}^{d_k} 1\cdot 1 = d_k
\qquad\text{(independence; } \Var=\E[\,\cdot\,^2]\text{ at mean }0).
$$

**Scaling.** $\Var$ is $2$-homogeneous, so dividing by $\sqrt{d_k}$ divides the variance by $d_k$:
$$
\Var\!\Big(\tfrac{\inner{\mathbf q}{\mathbf k}}{\sqrt{d_k}}\Big) = \tfrac{1}{d_k}\,\Var(\inner{\mathbf q}{\mathbf k}) = \tfrac{d_k}{d_k} = 1. \qquad\blacksquare
$$
</Theorem>

So $1/\sqrt{d_k}$ is the unique scale that holds logit variance at $1$ as $d_k$ grows — which is what keeps softmax gradients alive <Cite key="vaswani2017attention" />.

## Exercises

Attempt each before expanding its solution.

**Exercise 4.1 (Prove).** The weight matrix $\softmax(\mathbf Q\mathbf K^\top/\sqrt{d_k})$ is row-stochastic (<XRef id="def-row-stochastic" />); conclude each attention output is a convex combination of the value rows.

<SolutionBox>

Fix a query row $i$ and write the logits $\ell_{ij} = \inner{\mathbf q_i}{\mathbf k_j}/\sqrt{d_k}$. The attention weight $w_{ij} = e^{\ell_{ij}}/\sum_{j'} e^{\ell_{ij'}}$ is **positive** (an exponential over a positive sum) and the row **sums to one** ($\sum_j w_{ij} = \sum_j e^{\ell_{ij}} / \sum_{j'} e^{\ell_{ij'}} = 1$); so every row is a probability vector, i.e. the weight matrix is row-stochastic. The output row is $\sum_j w_{ij}\,\mathbf v_j$ with $w_{ij}\ge 0$ and $\sum_j w_{ij}=1$, i.e. a **convex combination** of the value rows. $\qquad\blacksquare$

</SolutionBox>

**Exercise 4.2 (Derive).** Without the $1/\sqrt{d_k}$ scaling, give $\Var(\inner{\mathbf q}{\mathbf k})$ and explain what happens to a softmax row as $d_k\to\infty$ (relate it to gradient saturation).

<SolutionBox>

By the variance-preservation theorem (unscaled case), $\Var(\inner{\mathbf q}{\mathbf k}) = d_k$. As $d_k\to\infty$ the logit spread grows without bound, so within a softmax row the largest logit dominates and the row concentrates toward a **one-hot** vector. At a near-one-hot $\mathbf p$, the softmax Jacobian $\operatorname{diag}(\mathbf p) - \mathbf p\mathbf p^\top$ has entries $J_{ij} = p_i(\delta_{ij} - p_j)$ (<XRef id="prop-softmax-jacobian" />) — diagonal $p_i(1-p_i)\to 0$ and off-diagonal $-p_i p_j\to 0$ — so gradients through the softmax **vanish** (gradient saturation). Dividing logits by $\sqrt{d_k}$ holds the variance at $1$ and prevents this. $\qquad\blacksquare$

</SolutionBox>

**Exercise 4.3 (Compute).** For $\mathbf z=(2,1,0)$, compute $\softmax(\mathbf z)$ and verify it is unchanged by $\mathbf z + 5\mathbf 1$.

<SolutionBox>

With $\mathbf z=(2,1,0)$ the unnormalized weights are $(e^2, e^1, e^0) \approx (7.389,\, 2.718,\, 1.000)$, summing to $\approx 11.107$, so
$$
\softmax(\mathbf z) \approx (0.665,\; 0.245,\; 0.090).
$$
For $\mathbf z + 5\mathbf 1 = (7,6,5)$ every weight is multiplied by $e^5$, which cancels in the normalization, giving the identical vector — exactly <XRef id="prop-softmax-translation-inv" />. $\qquad\blacksquare$

</SolutionBox>
