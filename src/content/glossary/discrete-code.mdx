---
term: "Code"
aliases: ["discrete code", "visual code", "code index"]
see: [token, unit, image-patch]
tags: [multimodal, tokenization]
---

A discrete index chosen from a learned codebook, especially by a visual quantizer. The **code** is the index $k$; the **code vector** $\mathbf e_k$ is its learned continuous representative. Once serialized into a sequence, a code is a token in the finite visual vocabulary.
